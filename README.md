# Predictive Analytics Using Historical Data

## Problem Statement
Build a predictive model to forecast future trends using historical data.

## Key Features
- Used regression and time-series models (ARIMA, Prophet)
- Cleaned and preprocessed historical datasets
- Evaluated model accuracy using MAE
- Visualized historical trends and future predictions

## Results
- **ARIMA MAE:** 399.41 ← Best Model
- **Prophet MAE:** 577.8

## Tech Stack
- Python (Pandas, NumPy, Statsmodels, Prophet)
- Google Colab
- Matplotlib for visualization

## Conclusion & Learnings
Learnt predictive modeling, trend analysis, and data-driven forecasting using time-series techniques.
